# ButtonLink

The `ButtonLink` is an extension of `ButtonBase` to support link styles.

### Size

Use the `size` prop and the `SIZES` object from `./ui/helpers/constants/design-system.js` to change the size of `ButtonLink`. Defaults to `SIZES.MD`

Optional: `BUTTON_LINK_SIZES` from `../../ui/components/component-library` object can be used instead of `SIZES`.

Possible sizes include:

- `SIZES.INHERIT` inherits the font-size of the parent element.
- `SIZES.SM` 32px
- `SIZES.MD` 40px
- `SIZES.LG` 48px

<Canvas>
  <Story id="ui-components-component-library-button-link-button-link-stories-js--size" />
</Canvas>

```jsx
import { SIZES } from '../../../helpers/constants/design-system';
import { ButtonLink } from '../../ui/components/component-library';

<ButtonLink size={SIZES.SM}>Small Button</ButtonLink>
<ButtonLink size={SIZES.MD}>Medium (Default) Button</ButtonLink>
<ButtonLink size={SIZES.LG}>Large Button</ButtonLink>
<ButtonLink size={SIZES.INHERIT}>Inherit Button</ButtonLink>
<ButtonLink noPadding>No Padding Button</ButtonLink>
```

### No Padding

Set `noPadding` to true when a `ButtonLink` for zero padding and when there isn't a `Text` parent component

No `size` prop is required since this will override the `ButtonLink` height to button text line height (22px)

<Canvas>
  <Story id="ui-components-component-library-button-link-button-link-stories-js--no-padding" />
</Canvas>

```jsx
import { ButtonLink } from '../../ui/components/component-library';

<ButtonLink noPadding>No Padding Button</ButtonLink>;
```

### Danger

Use the `danger` boolean prop to change the `ButtonLink` to danger color.

<Canvas>
  <Story id="ui-components-component-library-button-link-button-link-stories-js--danger" />
</Canvas>

```jsx
import { ButtonLink } from '../../ui/components/component-library';

<ButtonLink>Normal</ButtonLink>
<ButtonLink danger>Danger</ButtonLink>
```

### Href

When an `href` is passed the tag element will switch to an `anchor`(`a`) tag.

<Canvas>
  <Story id="ui-components-component-library-button-link-button-link-stories-js--href" />
</Canvas>

```jsx
import { ButtonLink } from '../../ui/components/component-library';

<ButtonLink href="/">Href Example</ButtonLink>;
```
